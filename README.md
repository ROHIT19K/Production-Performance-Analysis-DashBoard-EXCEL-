
# 📊 Production Performance Analysis Dashboard (Excel)

Interactive Excel Dashboard for analyzing production operations, manufacturing costs, regional production performance, and operational efficiency.

## 📌 Project Overview

The Production Performance Analysis Dashboard project focuses on analyzing manufacturing and production data using Microsoft Excel. The dashboard provides insights into production efficiency, operational costs, product-type performance, regional productivity, and workforce demographics.

The project helps manufacturing businesses track production metrics and optimize operational performance through interactive reporting.

## 🎯 Business Problem

Manufacturing organizations often struggle with:

* Total Production Cost By Product Type
* Number of Task By Each Manager
* Average Production Cost Per Unit By Production Type
* Total Units Produced By Year/Month 

This dashboard helps management monitor production KPIs and make data-driven operational decisions.
